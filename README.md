# 2D 슈팅
https://cafe.naver.com/onell0/40339 프로젝트에서 특정 컴에선 플레이를 못하는 치명적인 버그로 인해 새로만든 유니티 버전

2D 슈팅의 유니티 버전입니다

스크래치에서 특정 컴에선 플레이가 불가능한 치명적인 버그로

유니티로 새로 만들었습니다

스크래치랑 최대한 똑같이 만들었습니다

폰트는 못 구했습니다



혹시 버그 있으면 예기해주세요

Q 키로 수동으로 재장전 합니다

R 키를 길게 눌러 리스타트 합니다

WASD로 움직입니다

마우스 클릭으로 총을 발사합니다
